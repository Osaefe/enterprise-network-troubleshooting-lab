# Enterprise Network Troubleshooting & Infrastructure Lab

Designed and implemented a simulated enterprise LAN environment using Cisco Packet Tracer to demonstrate networking, infrastructure support, and fault-resolution capabilities.

## Project Scope
This lab simulates a medium-scale office network environment with segmented departments, shared services, and centralized connectivity.

## Key Features
- Router and switch configuration
- VLAN segmentation
- DHCP and DNS configuration
- IP address planning
- Static route implementation
- Shared printer and server access

## Skills Demonstrated
- Network troubleshooting
- Incident resolution
- Infrastructure diagnostics
